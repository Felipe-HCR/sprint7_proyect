# sprint7_proyect
Repositorio para el proyecto del sprint 7 de análisis de ventas de autos


Este proyecto genera una aplicación web la cual analiza en primer instancia el comportamiento del conjunto de datos de anuncios de venta de coches, mostrando un botón para crear un gráfico de dispersión o un histograma dependiendo de la casilla de verificación seleccionada, y que también mostrará un mensaje en caso de elegir ambas casillas o ninguna.

Además, aparecerá debajo del diagrama una ventana que muestra los datos registrados en el archivo de ventas de autos, permitiendo elegir la cantidad de filas con dos cuadros de texto y recorriendo dicha ventana por las filas deseadas.
